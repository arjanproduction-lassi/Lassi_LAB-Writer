# MVP Scope

## MVP Goal

Create the smallest useful Writer experience: a fast, mobile-first place to
capture and revisit one kind of creative spark.

The MVP should prove that Writer can be faster than forgetting.

## In Scope

- Mobile-first PWA shell.
- Fast creation of one text spark.
- Local-first spark persistence.
- Basic list and detail views.
- Reopen and edit a saved spark.
- Simple language that invites the author to describe the image, sound, feeling,
  or line that started the spark.

## Future Concepts To Preserve

These do not need to be fully implemented in the MVP, but the MVP should avoid
blocking them:

- Image-first capture.
- Voice sparks.
- Melody sparks.
- Draft shaping for poems and lyrics.
- Future AI helper Kováč.
- Export bridge to Songbook.
- Export bridge to Storyboard.
- File-based export.
- Offline-friendly usage.

## Out Of Scope For MVP

- Full app build before foundation is agreed.
- AI generation or editing.
- Songbook connection.
- Storyboard connection.
- Shared databases.
- Image upload, camera capture, or image library management.
- Voice recording or transcription.
- Melody recording or notation.
- Search and filtering.
- Dedicated poem, lyric, or draft editors.
- Export bridge implementation.
- Collaboration features.
- Account system.
- Cloud synchronization.
- Complex tagging taxonomy.
- Advanced publishing workflows.
- Final import/export schema.

## MVP Success Criteria

- A user can open Writer on a phone and capture a spark quickly.
- A user can return to recent sparks without friction.
- A user can reopen and edit a saved spark.
- The UI language respects that the spark may have started as an image, sound,
  melody, memory, or feeling.
- The app does not imply AI authorship.
- The storage approach keeps Writer independent from other LassiLAB tools.

## Product Risks

- Capture flow becomes too slow.
- The app becomes document-heavy too early.
- AI concepts overshadow the author's voice.
- Future integrations create pressure for a shared database.
- Data model becomes too rigid before real creative use is observed.
